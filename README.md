# Sokoban
Jeu du Sokoban réalisé en C jouable dans le terminal dans le cadre de la SAE 1.01, 

<img width="478" height="583" alt="image du jeu" src="https://github.com/user-attachments/assets/7a1c55fa-0154-4056-82de-c6d4ace259de" />

  
 Commandes :                              

- 'z' : haut 
- 's' : bas 
- 'q' : gauche   
- 'd' : droite 
- '+' : zoom              
- '-' : dezoom       
- 'u' : annuler le dernier déplacement
- 'r' : recommencer la partie
- 'x' : quitter le jeu
